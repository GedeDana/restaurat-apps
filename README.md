## How to use Website

1. _Clone Repository_ <br />
   ```
   git clone https://github.com/GedeDana/restaurat-apps.git
   ```
2. _Install Dependencies_ <br />
   ```
   npm install
   ```
3. run _project_ in terminal or text editor<br />
   ```
   npm run start-dev
   ```
4. Open url localhost with website
